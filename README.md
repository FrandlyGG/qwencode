# WB Monitor Bot

Telegram-бот для мониторинга свободных слотов отгрузки на складах Wildberries.

## 🚀 Возможности

- **Мониторинг складов**: Автоматическая проверка доступных слотов для отгрузки
- **Уведомления**: Мгновенные оповещения о новых слотах в Telegram
- **Гибкая настройка**: Выбор конкретных складов для мониторинга
- **Кэширование**: Оптимизация запросов к API Wildberries
- **Логирование**: Подробные логи для отладки и мониторинга
- **Статистика**: Отслеживание проверок, найденных слотов и уведомлений

## 📋 Требования

- Python 3.9+
- Токен API Wildberries (категория "Поставки")
- Токен Telegram-бота
- Chat ID для получения уведомлений

## ⚙️ Установка

1. Клонируйте репозиторий:
```bash
git clone <repository_url>
cd wb_monitor_bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Заполните `.env` своими данными:
```env
WB_API_TOKEN=ваш_токен_wildberries
TG_BOT_TOKEN=ваш_токен_telegram
TG_CHAT_ID=ваш_chat_id
CHECK_INTERVAL=60
MONITOR_WAREHOUSE_IDS=507,123456  # опционально
LOG_LEVEL=INFO
LOG_FILE=wb_monitor.log
```

## 🔑 Как получить токены

### Wildberries API Token
1. Зайдите в личный кабинет поставщика WB
2. Перейдите в раздел "Настройки" → "Доступ к API"
3. Создайте токен с правами доступа к категории "Поставки"

### Telegram Bot Token
1. Напишите @BotFather в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Сохраните полученный токен

### Chat ID
1. Напишите @userinfobot в Telegram
2. Бот отправит ваш Chat ID

## 🚀 Запуск

```bash
python wb_monitor_bot.py
```

## 📱 Использование

После запуска бота в Telegram:

1. Отправьте `/start` для главного меню
2. Нажмите "▶️ Запустить" или используйте `/start_monitoring`
3. Бот начнёт проверку складов и отправит уведомление при обнаружении слота

### Команды

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/start_monitoring` | Запустить мониторинг |
| `/stop_monitoring` | Остановить мониторинг |
| `/status` | Показать текущий статус |
| `/help` | Справка по боту |

## ⚙️ Настройка

### Мониторинг конкретных складов

В файле `.env` укажите ID складов через запятую:
```env
MONITOR_WAREHOUSE_IDS=507,123456,789
```

Оставьте пустым для мониторинга всех активных складов.

### Интервал проверки

Измените интервал проверки (в секундах):
```env
CHECK_INTERVAL=60
```

**Важно**: Не устанавливайте слишком маленький интервал! API Wildberries имеет лимиты (6 запросов в минуту на метод /warehouses).

## 📊 Логирование

Бот ведёт подробное логирование:
- Файл лога: `wb_monitor.log` (настраивается в `.env`)
- Уровни: DEBUG, INFO, WARNING, ERROR

Пример лога:
```
2024-01-15 10:30:00 - wb_monitor_bot - INFO - Инициализация сессии API Wildberries
2024-01-15 10:30:01 - wb_monitor_bot - INFO - Получено 45 складов
2024-01-15 10:30:05 - wb_monitor_bot - INFO - Найдено 3 доступных слотов
```

## 🛠️ Troubleshooting

### Ошибка "WB_API_TOKEN не установлен"
Проверьте, что токен правильно указан в `.env` файле.

### Бот не отправляет уведомления
1. Проверьте TG_CHAT_ID
2. Убедитесь, что бот добавлен в чат (если это групповой чат)
3. Проверьте права бота на отправку сообщений

### Rate limit от API WB
Увеличьте `CHECK_INTERVAL` до 60 секунд или больше.
