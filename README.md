# 🤖 Telegram Bot для мониторинга слотов Wildberries

Бот автоматически отслеживает доступные слоты для поставки товаров на склады Wildberries.

## 🔧 Исправленные ошибки (v2.0)

### ✅ Что исправлено в этой версии:

1. **Неправильный URL API** 
   - Было: `https://supply-wb.ru/api/v1/warehouses` (ошибка 404)
   - Стало: `https://supplies-api.wildberries.ru/api/v1/warehouses` (правильный URL)

2. **Ошибка SSL сертификата**
   - Добавлено отключение проверки SSL для отладки
   - Предупреждения urllib3 отключены

3. **Ошибка "chat not found"**
   - Бот теперь отправляет сообщения ТОМУ пользователю, который запустил команду
   - Не зависит от TG_CHAT_ID в конфиге

4. **Не показывал найденные слоты**
   - Добавлена фильтрация по городу
   - Улучшено логирование процесса поиска

5. **Клавиатура пропадала при вводе города**
   - После ввода города клавиатура возвращается автоматически
   - Добавлена кнопка "❌ Отмена"

6. **Unclosed session ошибка**
   - Добавлен правильный close сессии и бота при остановке

7. **Проверка токенов при старте**
   - Бот не запустится без токенов
   - Понятные сообщения об ошибке

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка токенов

Скопируйте `.env.example` в `.env` и заполните токены:

```bash
cp .env.example .env
```

Откройте `.env` и укажите:

```ini
# Токен бота от @BotFather
TG_BOT_TOKEN=1234567890:AABBccDDeeFFggHHiiJJkkLLmmNNooP

# Токен API Wildberries (категория "Поставки")
WB_API_TOKEN=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

# Уровень логирования
LOG_LEVEL=DEBUG
```

#### Где взять токены:

**Telegram Bot Token:**
1. Откройте @BotFather в Telegram
2. Создайте нового бота командой `/newbot`
3. Скопируйте полученный токен

**Wildberries API Token:**
1. Зайдите в личный кабинет селлера WB
2. Перейдите в раздел "Настройки" → "Доступ к API"
3. Создайте токен для категории **"Поставки"**
4. Скопируйте токен

### 3. Запуск бота

```bash
python wb_monitor_bot.py
```

## 📱 Использование в Telegram

### Команды:
- `/start` - Главное меню
- `/start_monitoring` - Запустить мониторинг
- `/stop_monitoring` - Остановить мониторинг
- `/status` - Показать статистику
- `/city` - Установить фильтр по городу
- `/help` - Справка

### Кнопки меню:
- **▶️ Запустить мониторинг** - Начало проверки складов
- **⏸️ Остановить мониторинг** - Остановка проверки
- **📊 Статус** - Показать статистику
- **🌍 Выбрать город** - Фильтр по городу (например, "Москва")
- **❓ Помощь** - Справка

### Пример работы:

1. Отправьте `/start` боту
2. Нажмите "🌍 Выбрать город"
3. Введите название города (например, `Москва`)
4. Нажмите "▶️ Запустить мониторинг"
5. Бот начнёт проверять склады каждые 30 секунд
6. При появлении свободных слотов вы получите уведомление

## 🔍 Как работает фильтрация по городу

Бот фильтрует склады по полям `name` и `address`:
- Если указан город "Москва", покажет только склады где в названии или адресе есть "москва"
- Если город не указан - показывает все склады
- Для сброса фильтра введите пустое сообщение

## 🐛 Troubleshooting

### Ошибка "Не указан токен"
Убедитесь, что файл `.env` существует и содержит правильные токены.

### Ошибка SSL
Для продакшена рекомендуется настроить правильные SSL сертификаты.
В текущей версии SSL проверки отключены для отладки.

### Бот не отправляет сообщения
Проверьте, что вы не заблокировали бота и он имеет права на отправку сообщений.

### HTTP 401 / 403
Неверный токен Wildberries. Проверьте, что токен действителен и имеет доступ к категории "Поставки".

### HTTP 429 (Too Many Requests)
Превышен лимит запросов к API WB. Бот автоматически ждёт 60 секунд при получении этой ошибки.
